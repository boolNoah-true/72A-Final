Programming

LeetCode
Leetcode is a site to practice and answer various coding questions often found in interviews.
Codeforces
Codeforces is similar to leetcode but with a focus on competitive programming contests
HackerRank
HackerRank is similar to the previous two sites but focusses on teaching you coding basics and helping you get certified in different areas of code.
